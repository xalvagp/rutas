# Rutas GPX

Colección de rutas GPS en formato GPX para ciclismo y actividades al aire libre en España.

## Descripción

Este repositorio contiene rutas GPS grabadas y planificadas en formato GPX (GPS Exchange Format), principalmente en la Comunidad de Madrid y otras regiones de España.

## Formato

Los archivos están en formato **GPX 1.1** y contienen:

- Coordenadas geográficas (latitud y longitud)
- Datos de altitud (metros)
- Segmentos de track con puntos de ruta detallados

## Uso

Para visualizar o usar estas rutas puedes utilizar aplicaciones como:

- [gpx.studio](https://gpx.studio) — editor web de rutas GPX
- [Komoot](https://www.komoot.com)
- [Strava](https://www.strava.com)
- [Garmin Connect](https://connect.garmin.com)
- [OsmAnd](https://osmand.net) (móvil)
- [Wikiloc](https://www.wikiloc.com)
